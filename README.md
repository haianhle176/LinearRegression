# Linear Regression

A high-performance Multivariate Linear Regression model built from scratch in C++.

## Features
- **Optimization:** AVX2 Intrinsics, FMA, and OpenMP.
- **Implementation:** Matrix operations and Gradient Descent from scratch.
- **Efficiency:** Cache-friendly memory access.

## Files
- `MLLIB.h/cpp`: Core library.
- `LinearRegression.cpp`: Main logic.
